Printer Preset

How it works

Configure your printer and filament settings, choose between linear DOE (Design of Experiments) or averaged retraction tests, and generate a test pattern.

  • Bed/Hotend Temp: Nozzle and bed temperatures for the test (in °C)
  • Extrusion Width & Layer Height: Affects line width and height of printed features
  • Pressure Advance: Klipper pressure advance (or M572 for Marlin) value
  • Filament Dia & Nozzle Dia: Used to calculate E values for correct extrusion
  • DOE Settings: Design of Experiments grid: rows, columns, and line count
